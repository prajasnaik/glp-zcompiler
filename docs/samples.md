# Samples Guide

The `samples/` folder is a feature regression pack for the language.

## Arithmetic and numbers

- `01_int_arithmetic.dpl` — integer arithmetic + power.
- `02_float_arithmetic.dpl` — float arithmetic path.
- `03_mixed_int_float.dpl` — mixed numeric expressions and promotion.
- `10_precedence.dpl` — precedence and right-associative power check.

## Booleans and comparisons

- `04_comparisons.dpl` — all comparison operators.
- `05_booleans.dpl` — `and`, `or`, and unary `!`.
- `14_combined_logic.dpl` — composed logic expression.

## Control flow

- `06_if_else.dpl` — if/else branch behavior.
- `07_if_no_else.dpl` — if without else.

## Loop and recurrence semantics

- `08_while_sum.dpl` — simple while with prime updates.
- `09_fibonacci.dpl` — recurrence update using simultaneous prime assignment.
- `11_nested_while.dpl` — nested loop behavior and scope interactions.
- `12_geometric_series.dpl` — iterative growth recurrence.
- `13_float_while.dpl` — float arithmetic in loop body.

## Functions

- `15_function_add.dpl` — typed function definition plus a direct call.
- `16_recursive_function_fib.dpl` — recursive function evaluation using implicit final-expression return.
